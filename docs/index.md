# Welcome to V-Insight Knowledge Base

Welcome to the internal R&D knowledge base for the **Deep Learning-Powered Visual SLAM** project. This repository serves as a central hub for all our findings, documentation, and technical resources.

## Project Overview

The goal of this project is to develop a Visual SLAM (Simultaneous Localization and Mapping) system specifically enhanced with Deep Learning techniques to assist in navigation for the visually impaired.

## Navigation

*   **[Project Info](project-info/tech-stack.md)**: Details about our current and recommended technology stack.
*   **[Research](literature.md)**: Literature review, papers, and theoretical background.
*   **[VSLAM Models](vslam/index.md)**: Analysis and resources regarding different SLAM implementations (ORB-SLAM3, Droid-SLAM, etc.).
*   **[Hardware](hardware/index.md)**: Specs and setup guides for our hardware platforms (Jetson, RPi, Cameras).
*   **[Datasets](datasets/index.md)**: Information suitable datasets for training and benchmarking.
