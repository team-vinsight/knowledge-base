# Jetson Orin NX

The NVIDIA Jetson Orin NX is a high-performance AI module designed for edge devices, delivering up to 100 TOPS of AI performance in a compact form factor.

## Technical Specifications

| Feature | Specification |
| :--- | :--- |
| **AI Performance** | 100 TOPS (16GB) / 70 TOPS (8GB) |
| **GPU** | NVIDIA Ampere architecture with 1024 CUDA cores and 32 Tensor Cores |
| **CPU** | 8-core (16GB) / 6-core (8GB) Arm® Cortex®-A78AE v8.2 64-bit CPU |
| **Memory** | 16GB or 8GB 128-bit LPDDR5 (102.4 GB/s) |
| **Power** | 10W - 25W (16GB) / 10W - 20W (8GB) |
