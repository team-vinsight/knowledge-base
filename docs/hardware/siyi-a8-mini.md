# SIYI A8 Mini

The SIYI A8 Mini is a compact 4K AI gimbal camera designed for drones and robotics, offering high-resolution video and stabilization.

## Technical Specifications

| Feature | Specification |
| :--- | :--- |
| **Gimbal** | 3-Axis Stabilization (Tilt, Roll, Pan) |
| **Sensor** | 1/1.7-inch Sony sensor (approx. 8MP effective) |
| **Video Resolution** | 4K Ultra HD (3840x2160 @ 30fps), 2K @ 30fps, 1080p @ 30/60fps |
| **Zoom** | 6x Digital Zoom |
| **Interfaces** | Ethernet (RTSP/UDP), HDMI, CVBS, S.Bus/UART |
| **Control** | SIYI Gimbal SDK / S.Bus / PWM |
